Token API

Aby uzyskać dostęp do naszych aplikacji mobilnych i do aplikacji stron trzecich, musisz najpierw wygenerować Token API.

  • Aby wygenerować token, po prostu kliknij przycisk „Utwórz”. Następnie skopiuj go i wklej wewnątrz aplikacji.
  • W zależności od funkcji, które chcesz udostępnić, wybierz rodzaj tokenu API, którego potrzebujesz.
